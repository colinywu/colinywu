Some Projects!

Initially being a simple computer science exercise in hash tables, the Employee Management System (EMS) eventually became a functional Java application developed in NetBeans which allows for a hypothetical company to manage employees. Initially, all the EMS did was sort employees into a hash table using their employee numbers and a corresponding hash function. However, the EMS is now quite versatile as it currently allows users to have password protection, add/remove/edit employees, add/remove/edit work locations, save/read information to/from text files, and navigate through their employees using a table and search bar.

The goal of this personal website (which you are currently viewing) was to present myself in a way more detailed than a social media profile or a resume. I intended for this website is one of the main showcases of my web design skills available on the Internet. Using pure HTML, CSS, and JS, I started creating the framework of the website. But as I went along, I added more advanced features such as responsiveness and click-interactions with jQuery and Bootstrap. As well, I learned more advanced technologies such as Jekyll/Liquid and SCSS to efficiently generate the static websites and easily add styling features. With Liquid, I was able to add logic and begin rudimentary interactions with databases (in the form of .yml files). I employed these technologies in this website to make as clear a presentation of myself as possible to the world. While some finishing touches need to be added, this website has proven to be effective in accomplishing this purpose. As well, I have been able to add blog functionalities through Jekyll, so that I am able to add, remove, and manage posts that express my personal opinions. Overall, the site is fully responsive and interactive, with drop-down menus, routing, and screen-size adaptbility. The process of designing and publishing this website has also added to my web design skills and furthered my interest in this technological area.

Beginning to learn JavaScript, I decided to develop a blackjack card game for the web. Implementing simple AI logic for the dealer, this game is fully functioning with one other player. Structured with simple HTML and styled with basic CSS, the game is founded upon the logic and programming in JavaScript. This game has peaked my interest into AI and machine learning and also led me into the rapid and fast growing reliance on JS as the way of interacting with the web. Since then, I have begun my learnings into React and continued developing my programming skills for the purposes of creating fully-functioning web and mobile apps.
